Frequently Asked Questions

Q: Will this extension always work, or will Chrome remove it?

A: The extension will continue to work indefinitely as long as you keep Developer Mode enabled. Chrome will NOT automatically remove it. You may see a notification about "Developer mode extensions" when Chrome starts‚Äîthis is normal. Just dismiss it.

Q: Is it safe to enable Developer Mode?

A: Yes, it's completely safe. Developer Mode simply allows you to install extensions from local files instead of only from the Chrome Web Store. It doesn't reduce your browser's security. Just don't install extensions from sources you don't trust.

Q: Can you publish this to the Chrome Web Store so I don't have to do this?

A: Yes! Publishing to the Chrome Web Store is on the roadmap for future versions. For now, this manual installation method ensures you can start using Unbreakable immediately while maintaining the same privacy guarantees.

Q: Does the extension still make zero network requests?

A: Absolutely! The extension has the exact same privacy guarantees as the web app. It runs 100% on your device and makes ZERO network requests. You can verify this yourself using Chrome's DevTools Network tab.

Q: What if I see a warning about the extension?

A: Chrome may show a warning banner about "extensions running in developer mode" when you start your browser. This is normal and safe to dismiss. The warning appears for ALL locally installed extensions, not because there's anything wrong with Unbreakable.

Q: Can I use both the extension and the web app?

A: Yes! Use whichever is more convenient at the moment:
‚Ä¢ Extension: Quick access from any webpage
‚Ä¢ Web App: Full-screen interface with more options
Both are completely offline and maintain the same security standards.
